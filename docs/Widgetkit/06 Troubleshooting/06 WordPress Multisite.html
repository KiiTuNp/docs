WordPress Multisite with Widgetkit 2

1. How do I install a Multisite with Widgetkit 2?

First you need to activate the Multisite in WordPress. To do that access your site's root via FTP and open the file wp-config.php. Look for the line:


		/* That's all, stop editing! Happy blogging. */

and add the following code above it:


		define('WP_ALLOW_MULTISITE', true);

Go back to your WordPress Dashboard and deactivate every plugin you have. Next go to _Tools -> Network Setup_ and follow the instructions.

For a complete instruction on how to install a Multisite please have a look at the official WordPress Codex: [Create a Multisite](http://codex.wordpress.org/Create_A_Network)
After you added the code snippets to your wp-config.php and .htaccess file, reload the page and log yourself back in. Now you installed the Multisite. Create a new Site by going to Sites -> Add new_.
To activate Widgetkit on your Sites you can not use the Network Activate Bulk Action. Widgetkit need its own database within the Multisites so you have to activate Widgetkit on every site by itself.

Go to every Site you installed and activate Widgetkit.

And there you have it. Now you can create Widgets with Widgetkit on every Multisite.

2. When I click on Widgetkit in my Multisite I get the Message 'Insufficient User Rights', why?

If you get the Error Message you activated Widgetkit in your Network Admin via the Bulk Option Network Activate which does not work with Widgetkit. Deactivate Widgetkit in your Network Admin and go to your Multisite. Click on Plugins and activate Widgetkit right there for every site.

3. The Widgets I created with Widgetkit won't save, why?

This could also happen if you activated Widgetkit as the Network Admin. Deactivate Widgetkit and go to every Multisite you created to activate Widgetkit one by one.
