Joomla! Cache Issues

Widgetkit 2 is not compatible with Joomla! Progressive Cache because in such environment some necessary events are not getting triggered. If your project requires the Caching feature there is a workaround with a custom script.

Place the following code in your Theme custom PHP script file or withing a custom system plugin:

if ($app = @include JPATH_ADMINISTRATOR.'/components/com_widgetkit/widgetkit-app.php') { $app->trigger('view', array($app)); }
